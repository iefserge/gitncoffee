# AI promise

Git ‘n’ Coffee does not use or share repositories to train machine learning models. But, of course,
public repositories can be downloaded by external tools in any case.

This policy is written to [Git ‘n’ Coffee Terms or Service](https://gitncoffee.com/terms) and will not change.

## Code Review

There are no plans for this yet, but Git ‘n’ Coffee may decide in the future to offer
some type of AI-assisted code review. It will always be opt-in and will require explicit approval to enable.
